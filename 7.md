# Open Source GEO (Generative Engine Optimization) Tools

## GEO & AI Search Optimization
- **[AutoGEO](https://github.com/topics/generative-engine-optimization)**: Framework for automatically optimizing content for generative search engines.
- **[GetCito](https://github.com/topics/generative-engine-optimization)**: Open Source AI Search Optimization (AIO) / Answer Engine Optimization (AEO) tool.
- **[AI Monitor](https://github.com/topics/generative-engine-optimization)**: Tracks brand mentions in AI answers (ChatGPT, Gemini, Perplexity).

## LLM Optimization & Content Strategy
- **[RivalSee AI SEO](https://github.com/topics/ai-seo)**: Collection of open-source AI SEO tools for modern web apps.
- **[SEOMachine](https://github.com/topics/ai-seo)**: AI content writer for researching and outlining articles to target AI answer engines.
- **[llms.txt Generators](https://github.com/topics/llms-txt)**: Various scripts to generate `llms.txt` files, helping LLMs understand your website structure.

## Prompt Engineering for SEO
- **[PromptLayer](https://github.com/PromptLayer/promptlayer)**: Middleware for prompt engineering, useful for testing and optimizing SEO prompts for content generation.
- **[LangChain](https://github.com/langchain-ai/langchain)**: Key framework for building LLM applications, useful for creating custom SEO agents.
