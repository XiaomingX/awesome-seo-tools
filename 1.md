# Build your own Semrush

## Rank Tracking
- **[SerpBear](https://github.com/towfiqi/serpbear)**: Open Source Search Engine Position Tracker.
  - Features: Unlimited keywords, Google Search Console integration, email notifications.

## Site Audit & Technical SEO
- **[Google Lighthouse](https://github.com/GoogleChrome/lighthouse)**: Automated tool for improving the quality of web pages.
- **[SEO Macroscope](https://github.com/nazuke/SEOMacroscope)**: Website broken link checker and site scanner.
- **[SEO Titan](https://github.com/D4Vinci/SEO-Titan)**: Automated SEO analysis tool in Python.

## Web Analytics
- **[Matomo](https://github.com/matomo-org/matomo)**: The leading open-source web analytics platform that gives you 100% data ownership.
- **[Umami](https://github.com/umami-software/umami)**: Simple, privacy-focused open-source web analytics.

## Backlink Analysis
- **[Screaming Frog](https://www.screamingfrog.co.uk/seo-spider/)** (Free version available, extensive use in industry): Website crawler for technical SEO.
- **[LinkChecker](https://github.com/linkchecker/linkchecker)**: Check websites and HTML documents for broken links.
