# Open Source Replit Alternatives (Online IDEs)

Self-hosted and cloud-based development environments to code from anywhere.

## Online Development Environments
- **[Coder](https://github.com/coder/coder)**: **(Self-Hosted)** Provision remote development environments on your own infrastructure (Kubernetes, Docker, etc.).
- **[Eclipse Che](https://github.com/eclipse-che/che)**: Kubernetes-native IDE and developer workspace server.
- **[Gitpod](https://github.com/gitpod-io/gitpod)**: Automated, ephemeral development environments in the cloud. Defined as code (`.gitpod.yml`).
- **[CodeSandbox](https://github.com/codesandbox/codesandbox-client)**: Instant cloud development environments, particularly strong for web frameworks.
- **[Theia](https://github.com/eclipse-theia/theia)**: Cloud & desktop IDE framework. It's the open-source tech behind many custom cloud IDEs.

## Web-Based Editors
- **[VSCodium](https://github.com/VSCodium/vscodium)**: Free/Libre Open Source Software Binaries of VS Code, can be run in browser environments via OpenVSCode Server.
- **[OpenVSCode Server](https://github.com/gitpod-io/openvscode-server)**: Run VS Code on a remote machine and access it in a browser.
