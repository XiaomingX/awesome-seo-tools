# Open Source Retool Alternatives (Internal Tools)

These low-code platforms allow you to build custom admin panels, dashboards, and internal tools by connecting to your databases and APIs.

## Top Recommendations
- **[Appsmith](https://github.com/appsmithorg/appsmith)**: **(Most Popular)** Developer-first open-source platform to build enterprise internal tools. Drag & drop UI with JS logic.
- **[ToolJet](https://github.com/ToolJet/ToolJet)**: Extensible low-code framework to build business applications. Supports Python/JS and has extensive integrations.
- **[Budibase](https://github.com/Budibase/budibase)**: Focuses on speed and ease of use. Great for building internal apps and workflows on top of your own data.
- **[Lowdefy](https://github.com/lowdefy/lowdefy)**: Build internal tools, web apps, and admin panels using YAML. Best for config-driven development.
- **[ILLA Cloud](https://github.com/illacloud/illa-builder)**: Designed for developers, offering powerful AI coding assistance and real-time collaboration.

## Other Worthy Mentions
- **[Openblocks](https://github.com/openblocks-dev/openblocks)**: Open-source Retool alternative, visual programming.
- **[Refine](https://github.com/refinedev/refine)**: React-based framework for building internal tools, though more code-heavy than pure drag-and-drop builders.
