# Internal Tools and Online IDEs

## Low-Code Internal Tool Builders
Build custom admin panels, dashboards, and internal tools by connecting to your databases and APIs.

- **[Appsmith](https://github.com/appsmithorg/appsmith)**: Developer-first platform to build enterprise internal tools. Drag & drop UI with JS logic.
- **[ToolJet](https://github.com/ToolJet/ToolJet)**: Extensible low-code framework supporting Python/JS and extensive integrations.
- **[Budibase](https://github.com/Budibase/budibase)**: Focuses on speed and ease of use for building internal apps and workflows.
- **[Lowdefy](https://github.com/lowdefy/lowdefy)**: Build internal tools and admin panels using YAML. Best for config-driven development.
- **[ILLA Cloud](https://github.com/illacloud/illa-builder)**: Designed for developers, offering AI coding assistance and real-time collaboration.

## Online Development Environments (IDEs)
- **[Coder](https://github.com/coder/coder)**: Provision remote development environments on your own infrastructure (Kubernetes, Docker).
- **[Eclipse Che](https://github.com/eclipse-che/che)**: Kubernetes-native IDE and developer workspace server.
- **[Gitpod](https://github.com/gitpod-io/gitpod)**: Automated, ephemeral development environments in the cloud defined as code.
- **[CodeSandbox](https://github.com/codesandbox/codesandbox-client)**: Instant cloud development environments, particularly strong for web frameworks.
- **[Theia](https://github.com/eclipse-theia/theia)**: Cloud & desktop IDE framework behind many custom cloud IDEs.

## Web-Based Editors
- **[VSCodium](https://github.com/VSCodium/vscodium)**: Free/Libre binaries of VS Code.
- **[OpenVSCode Server](https://github.com/gitpod-io/openvscode-server)**: Run VS Code on a remote machine and access it in a browser.
